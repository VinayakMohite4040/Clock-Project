Clock-project
-------------------------------------------------------------------------------------------------------------
A simple app that lets you to view the time in 5 different MathWorks office locations


-------------------------------------------------------------------------------------------------------------
Description
-------------------------------------------------------------------------------------------------------------
User can view time in the following locations:
  1. Natick, MA USA
  2. Galway, Ireland
  3. Beijing, China
  4. Hyderabad, India
  5. Chatswood, Australia
